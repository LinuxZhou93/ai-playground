# 赛车小游戏

一个使用 Python Pygame 开发的简单赛车游戏。

## 游戏特性
- 使用方向键或 WASD 控制赛车
- 躲避迎面而来的敌车
- 实时计分系统
- 碰撞检测
- 游戏难度随时间递增

## 安装依赖
```bash
pip install -r requirements.txt
```

## 运行游戏
```bash
python main.py
```

## 游戏操作
- **上箭头/W**: 加速
- **下箭头/S**: 减速
- **左箭头/A**: 向左转
- **右箭头/D**: 向右转
- **ESC**: 退出游戏

## 游戏规则
1. 控制你的赛车避开迎面而来的敌车
2. 每成功避开一辆敌车得 10 分
3. 撞到敌车游戏结束
4. 难度随时间递增，敌车速度会越来越快
